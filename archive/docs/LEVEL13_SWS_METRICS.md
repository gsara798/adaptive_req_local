# Level 13 SWS performance metrics

Level 13 analyzes the out-of-sample SWS predictions generated by Level 12.
It intentionally uses the test split only. Training conditions are excluded
from performance claims because their errors are optimistic estimates of
generalization.

Level 13 also writes descriptive `all_conditions` tables using train and test
predictions. These tables provide MAPE for all simulated conditions, but they
must not be interpreted as unbiased generalization metrics.

## Analysis hierarchy

### Patch prediction

One predicted SWS value for one local patch.

### Intramap unit

One fixed:

```text
model + condition + aperture step + realization
```

The patches inside that realization are treated as spatial samples from one
homogeneous estimated SWS map.

Intramap metrics include:

- `map_APE_mean_pct`: accuracy of the map mean.
- `map_APE_median_pct`: accuracy of the map median.
- `patch_MAPE_pct`: average patch-level absolute percentage error.
- `patch_RMSE_pct`: patch-level percentage RMSE.
- `bias_pct`: signed percentage error.
- `CoV_pct`: within-map coefficient of variation.
- `IQR_over_median_pct`: robust within-map precision metric.
- `valid_fraction`: fraction of finite SWS predictions.

### Interseed unit

One fixed:

```text
model + condition + aperture step
```

Intramap metrics are summarized across `realization_idx`.

Interseed metrics include:

- Median, IQR, standard deviation, P90, P95, and worst intramap APE.
- Robust relative APE variability: `100 * IQR / abs(median)`.
- Interseed CoV of the map-mean SWS.
- Failure rate.

### Condition unit

One fixed:

```text
model + condition_id
```

A condition is defined by `WaveModel`, `f0`, true SWS, and `REQ_M`, and
contains all aperture steps, realizations, and patches available in the test
split.

Condition metrics include MAPE, MdAPE, percentage RMSE, bias, P90/P95 error,
intramap precision, and interseed stability.

The factor boxplot labeled `Patch-level absolute SWS error` contains every
individual patch prediction. Outliers can therefore be much larger than the
MAPE of a condition, which is the mean absolute percentage error across all
patches, aperture steps, and realizations in that condition.

## Accuracy, precision, and robustness

- **Accuracy**: low MAPE, MdAPE, RMSE, and absolute bias.
- **Intramap precision**: low CoV and IQR-to-median ratio.
- **Interseed robustness**: low IQR/P90/P95 across realizations and low
  interseed CoV.

These objectives should be inspected together. A model with the lowest global
MAPE may still be unstable for specific apertures or physical scenarios.

## Diagnostic models

Models containing `diagnostic` in their name are reported separately.
`ModelD_diagnostic_with_Meff` uses `M_eff`, which depends on the true SWS, so
it is useful as an upper-bound diagnostic but cannot be deployed on unknown
experimental data.
