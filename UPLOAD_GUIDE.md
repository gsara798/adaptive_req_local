# Upload guide

Place these files in the root of `gsara798/adaptive_req_local`:

- `README.md`
- `CITATION.cff`
- `CONTRIBUTING.md`
- `CHANGELOG.md`
- `CODE_OF_CONDUCT.md`

Keep the `LICENSE` file that is already present.

Recommended commit message:

```text
Add open-source project documentation
```

Before creating release `v0.1.0`, replace `YYYY-MM-DD` in `CHANGELOG.md` with
the release date and confirm that at least one public example runs from a clean
clone.
